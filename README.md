# 🍲 Gin Rai Dee (กินไรดี) | Food Random Web Application

แอปพลิเคชันสุ่มเมนูอาหารเพื่อแก้ปัญหา "วันนี้กินอะไรดี?" โดยเน้นเมนูอาหารท้องถิ่นและการใช้งานที่รวดเร็ว

## ✨ ฟีเจอร์เด่น
- **Dynamic Category Filtering:** เลือกระบุหมวดหมู่ได้ (เช่น ก๋วยเตี๋ยว, อาหารตามสั่ง)
- **Smart Recommendation:** แสดงผลการสุ่มพร้อมรายละเอียด, แคลอรี่โดยประมาณ และช่วงราคา
- **Minimalist UI/UX:** ออกแบบหน้าตาให้สะอาดตา ใช้งานง่าย รองรับการแสดงผลทุกหน้าจอ (Responsive)

## 🛠️ Tech Stack
- **Frontend:** React / JavaScript
- **Styling:** Tailwind CSS
- **Version Control:** Git & GitHub
